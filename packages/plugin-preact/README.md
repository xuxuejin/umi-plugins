# @umijs/plugin-preact

> @umijs/plugin-preact.

See our website [@umijs/plugin-preact](https://umijs.org/plugins/plugin-preact) for more information.

## Install

Using npm:

```bash
$ npm install --save-dev @umijs/plugin-preact
```

or using yarn:

```bash
$ yarn add @umijs/plugin-preact --dev
```
